# Learning-AndroidDevelopment

本项目是为初级Android开发提供系统的Android开发指导，主要分如下几个模块：

### Android应用开发核心技术介绍
   
   以专题方式来研究Android的核心技术

- Android四大组件及通信框架
- Android UI设计技术
- Android的异步技术(多线程编程)
- Android动画
- Android网络编程及框架(框架)
- Android数据存储(含ORM框架)
- MVC与MVP的框架设计

### Framework层核心服务
   该模块是探讨AMS，PMS等支持上层应用开发工作原理，能够使得大家更加深入理解
开发过程

### 不同类别应用产品分析研究

　　本模块主要是分别研究不同类别的应用开发模型和技术选型

### 资源引用

    
本章节将罗列出活跃与社区的开发者和一些开源资源库的，使得能够跟随最新的技术趋势，更上一层楼
